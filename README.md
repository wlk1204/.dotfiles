# dotfiles
Dot files of Tools for coding.



### iterm2
```bash
brew install iterm2
```



### zsh
```bash
# curl
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```



### neovim
```bash
brew install neovim
nvim -v
```



### tmux
```bash
brew install tmux
tmux -V
```



### nerd-fonts
```bash
brew tap homebrew/cask-fonts
brew cask install font-hack-nerd-font
```
